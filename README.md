# Taxidermy Education Website

An educational web application built using Python and Streamlit that provides information, tutorials, and resources about the art and science of taxidermy.

## Overview

Taxidermy Education is a simple learning platform designed to help beginners and enthusiasts understand the basics of taxidermy including history, tools, legal considerations, and practical techniques.

The website also includes image galleries and video tutorials showcasing taxidermy work.

## Features

- Educational content about taxidermy
- Image gallery of taxidermy projects
- Video tutorials
- Simple navigation menu
- Clean and responsive interface
- Easy deployment using Streamlit Cloud

## Technologies Used

- Python
- Streamlit
- GitHub
- Streamlit Community Cloud

## Installation

1. Clone the repository

```
git clone https://github.com/shafiq73/taxidermy-education.git
```

2. Install requirements

```
pip install -r requirements.txt
```

3. Run the app

```
streamlit run app.py
```

## Deployment

This project can be deployed easily using Streamlit Community Cloud.

Steps:

1. Push the project to GitHub
2. Go to Streamlit Cloud
3. Connect your GitHub repository
4. Deploy the app
